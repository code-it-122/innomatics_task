# Food Delivery Data Analysis

## Overview
This project merges food delivery data from multiple sources to create a single dataset for analysis.  
The final dataset is used to study orders, users, revenue, and restaurant performance.

## Data Sources
- `orders.csv` – Order transactions  
- `users.json` – User details and membership  
- `restaurants.sql` – Restaurant details (cuisine, rating)

## Tools Used
Python, Pandas, SQLite, Google Colab

## Process
- Loaded CSV, JSON, and SQL data  
- Merged datasets using LEFT JOIN  
- Created `final_food_delivery_dataset.csv`

## Output
- `final_food_delivery_dataset.csv`
